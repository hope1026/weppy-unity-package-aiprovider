# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-01

### Added

- Initial public release of `com.weppy.aiprovider`.
- Unified provider architecture for AI features.
- Chat support for OpenAI, Google Gemini, Anthropic Claude, and OpenRouter.
- Image generation support for OpenAI and Google Gemini models.
- Background removal support via Remove.bg integration.
- Unity Editor tools for provider setup, model selection, and request testing.
- Runtime API and sample assets for integration and verification.
