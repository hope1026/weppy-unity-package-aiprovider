# Editor Window

Unity エディター内でチャットを直接テストできます。

## 開く

1. Unity メニューで **Window > Weppy > AI Provider Chat** を開きます。
2. 上部タブを使います。
   - `Settings`: API キー、モデル既定値、各種オプション
   - `Chat`: プロンプト送信と応答確認

## 基本フロー

1. `Settings` で少なくとも 1 つのプロバイダーを設定します。
2. `Chat` タブに移動します。
3. プロバイダー/モデルを選択して送信します。
4. 応答と使用量情報を確認します。

## 注意

- Play モードに入らず利用できます。
- このウィンドウはチャット用途専用です。
