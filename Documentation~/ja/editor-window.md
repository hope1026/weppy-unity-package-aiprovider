# エディターウィンドウ

Unity エディター内で機能を直接テストできます。

## エディターウィンドウを開く
1.  Unity メニューで **Window > Weppy > AI Provider** を選択します。
2.  上部タブ（`Chat`、`Image`、`BgRemoval`）で各機能をテストします。

## 基本的な流れ
1.  各タブの Provider セクションで API キーを入力します。
2.  モデルを選択するか、カスタムモデル ID を保存します。
3.  リクエストを入力して実行し、結果を確認します。

## 注意事項
- エディターウィンドウは Play モードに入らずに利用できます。
