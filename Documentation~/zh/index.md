# Weppy AI Provider 文档

欢迎使用 **Weppy AI Provider** 文档。本包为 Unity 项目提供统一接口，以集成多种 AI 提供商。

## 概览

Weppy AI Provider 简化了连接多个 AI 服务（聊天、图像生成、背景去除）的流程。内部处理不同提供商的 API 差异，因此只需改动一行代码或设置即可切换提供商。

## 目录

- **[快速开始](getting-started.md)**：安装、设置与首次 API 调用。
- **[聊天](chat.md)**：聊天 API 使用指南（OpenAI、Gemini、Claude 等）。
- **[图像生成](image-generation.md)**：使用 DALL-E、Imagen 等生成图像。
- **[背景去除](bg-removal.md)**：背景去除功能使用方法。
