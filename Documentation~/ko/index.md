# Weppy AI Provider 문서

**Weppy AI Provider** 문서에 오신 것을 환영합니다. 이 패키지는 다양한 AI 공급자(Provider)를 유니티 프로젝트에 통합하기 위한 단일화된 인터페이스를 제공합니다.

## 개요

Weppy AI Provider는 채팅, 이미지 생성, 배경 제거 기능을 위해 여러 AI 서비스에 연결하는 과정을 단순화합니다. API 차이점을 내부적으로 처리하므로, 코드 한 줄이나 설정 변경만으로 공급자(Provider) 간 전환이 가능합니다.
