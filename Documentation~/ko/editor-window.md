# 에디터 창

Unity 에디터에서 채팅을 바로 테스트할 수 있습니다.

## 열기

1. Unity 메뉴에서 **Window > Weppy > AI Provider Chat**을 엽니다.
2. 상단 탭을 사용합니다.
   - `Settings`: API 키, 모델 기본값, 옵션 설정
   - `Chat`: 프롬프트 전송 및 응답 확인

## 기본 흐름

1. `Settings`에서 최소 1개 공급자를 설정합니다.
2. `Chat` 탭으로 이동합니다.
3. 공급자/모델을 선택하고 메시지를 전송합니다.
4. 응답과 사용량 정보를 확인합니다.

## 참고

- Play 모드 진입 없이 사용할 수 있습니다.
- 이 창은 채팅 워크플로우 전용입니다.
