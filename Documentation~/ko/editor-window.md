# 에디터 창 사용법 (Editor Window)

유니티 에디터에서 바로 기능을 테스트할 수 있습니다.

## 에디터 창 열기
1.  Unity 메뉴에서 **Window > Weppy > AI Provider**를 선택합니다.
2.  상단 탭에서 `Chat`, `Image`, `BgRemoval`을 선택해 기능별로 테스트합니다.

## 기본 사용 흐름
1.  각 탭의 Provider 섹션에서 API 키를 입력합니다.
2.  모델을 선택하거나 커스텀 모델 ID를 입력해 저장합니다.
3.  요청을 입력하고 실행하여 결과를 확인합니다.

## 참고 사항
- 에디터 창은 런타임 Play 모드 없이도 테스트가 가능합니다.
