# Editor Window

You can test chat directly inside the Unity Editor.

## Open

1. In Unity, open **Window > Weppy > AI Provider Chat**.
2. Use the top tabs:
   - `Settings`: configure provider keys, model preferences, and options.
   - `Chat`: send prompts and inspect responses.

## Workflow

1. Configure at least one provider in `Settings`.
2. Move to `Chat`.
3. Select provider/model and send a prompt.
4. Review responses and usage info.

## Notes

- The window works without entering Play mode.
- This editor tool is for chat workflows only.
