# Editor Window

You can test features directly inside the Unity Editor.

## Open the Editor Window
1.  In the Unity menu, select **Window > Weppy > AI Provider**.
2.  Use the top tabs (`Chat`, `Image`, `BgRemoval`) to test each feature.

## Basic Workflow
1.  Enter API keys in the Provider section of each tab.
2.  Select a model or save a custom model ID.
3.  Enter your request and run it to see results.

## Notes
- The editor window can be used without entering Play mode.
